# 小白执行手册（Agent2 导入）

## 1) 先决条件
- Edge 已用调试模式启动：9223
- 已登录 https://cst.uf-tree.com
- 拿到 Agent1 产物：`.xlsx` 三表文件

## 2) 先预检
```bash
python3 scripts/preflight_check.py
```
全绿后再执行。

## 3) 一键导入
```bash
python3 scripts/import_from_agent1.py \
  --xlsx "/path/to/三表联动_客户模板_xxxx.xlsx" \
  --output "./agent2_import_report.json"
```

## 4) 导入顺序（固定）
1. Sheet1 员工导入
2. Sheet2 科目 + 费用角色链路
3. Step2.5 审批流 workflowId 准备
4. Sheet3 模板创建 + 费用限制分支

## 5) 关键规则
- 借款单/申请单：不进入费用限制分支
- 报销单/批量付款单：按条件二选一
  - `归属单据名称` + `单据适配人员` 同时非空：费用角色限制分支
  - 否则：直接限制末级科目分支
